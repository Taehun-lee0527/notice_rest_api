# 실행 환경
* APPLE MacOS - Sonoma 14.5
* intelliJ 2024.02
* MySQL 8.3.0
* Spring boot 3.3.3, openJDK 17, JPA, queryDSL

----
# 환경 설정
* mac brew 설치
  * <https://brew.sh/ko/>
* openJDK 설치
  * brew install openjdk@17
* mysql 설치
  * brew install mysql
  * mysql --version
  * mysql.server start
  * mysql workbench 설치 <https://dev.mysql.com/downloads/workbench/>
* 프로젝트 실행 후 <localhost:8080/login> 에서 계정 생성 및 접속
